# [RSA-cryptographer](https://github.com/dudushy/RSA-cryptographer/)
Cryptography application made in Angular.

---
## Prerequisites:
- [NodeJS](https://nodejs.org/)

## How to install and run:
1. Install packages
    ```bash
    npm i
    ```
2. Run project
    ```bash
    npm start
    ```
3. Build project
    ```bash
    npm run build
    ```
4. Deploy project (gh-pages)
    ```bash
    npm run deploy
    ```
